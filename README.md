# Airline Passenger Satisfaction Prediction ✈️

# Objective

The goal of this project is to predict airline passenger satisfaction based on travel experience and in-flight service features. The prediction helps airlines identify areas for improvement and enhance customer satisfaction.

# Dataset Used

Name: Airline Passenger Satisfaction Dataset

Source: Kaggle – Airline Passenger Satisfaction

Description: The dataset contains passenger demographics, flight details, and ratings for in-flight services. Key features include Gender, Age, Flight Distance, Seat Comfort, In-flight Entertainment, Food & Beverages, Online Booking Experience, Departure/Arrival Delays, and the target variable Satisfaction (Satisfied / Dissatisfied).

# Models Used

Random Forest Classifier – Handles non-linear relationships well, robust to overfitting, and works with mixed feature types.

These models were chosen to compare performance across interpretable, ensemble, and boosting techniques for classification tasks.

# Key Results

--- Random Forest Performance ---
Accuracy : 0.9471
Precision: 0.9485
Recall   : 0.9291
F1-Score : 0.9387

Outcome: Random Forest Classifier achieved the highest accuracy (94%), F1 score (0.93) making it the most effective in predicting passenger satisfaction.
